# Contributing

Takk for bidrag til Ønsketur.

## Lokal utvikling

1. Klon repoet.
2. Start lokal server i prosjektmappen, for eksempel:

   ```bash
   python -m http.server 8080
   ```

3. Åpne `http://localhost:8080` i nettleser.

## Retningslinjer

- Hold endringer små og fokuserte.
- Bevar mobil-først-oppsett.
- Unngå nye tunge avhengigheter uten tydelig behov.
- Test manuelt i mobilvisning før du oppretter PR.

## Commit og PR

- Bruk beskrivende commit-meldinger.
- Beskriv hva som er endret og hvordan det kan verifiseres.
- Legg ved skjermbilder ved UI-endringer.
